# Frontend

React + Vite + TypeScript frontend for AI PR Review Assistant.

PR 1 includes only the workbench shell. API integration, result rendering, Markdown copy, warning states, and localStorage history will be added in later PRs.

## Local Development

```powershell
npm install
npm run dev
```

Run tests:

```powershell
npm test
```

Build:

```powershell
npm run build
```
